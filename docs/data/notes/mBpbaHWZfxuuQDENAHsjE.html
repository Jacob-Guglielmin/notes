Basic Reaction Types

Formation: A + B → AB

Decomposition: AB → A + B

Single Replacement: A + BC → AB + C

Double Replacement: AB + CD → AD + CB

Combustion: C( )H( ) + O2 → CO2 + H2O

Reactions Practice

  1. Sulfurous acid is mixed with aqueous sodium hydroxide.

    H2SO3(aq) + 2NaOH(aq) → Na2SO3(aq) + 2H2O (Double Replacement)

  2. Copper metal is added to gold (III) chloride solution.

    3Cu(s) + 2AuCl3(aq) → 3CuCl2(aq) + 2Au(s) (Single Replacement)

  3. Carbon dioxide reacts with water to produce carbonic acid.

    CO2(g) + H2O → H2CO3(aq) (Formation)

  4. Sucrose is burned.

    C12H22O11(s) + 12O2(g) → 12CO2(g) + 11H2O (Combustion)

  5. Aqueous calcium hydroxide reacts with hydrochloric acid.

    Ca(OH)2(aq) + 2HCl(aq) → CaCl2(aq) + 2H2O (Double Replacement)

  6. Iron reacts with oxygen.

    4Fe(s) + 3O2(g) → 2Fe2O3(s) (Formation)

  7. Calcium chloride in solution is added to aqueous aluminum sulfate.

    3CaCl2(aq) + Al2(SO4)3(aq) → 3CaSO4(aq) + 2AlCl3(aq) (Double Replacement)
